02 — Herramientas
🌊
DigitalOcean

Droplets para backends y APIs. Sencillo de administrar, predecible en costos y con buen rendimiento para proyectos de tamaño medio.

☁️
AWS EC2

Instancias para aplicaciones web. Flexibilidad en tipo de instancia, integración con otros servicios de AWS y control total del servidor.

🐳
Docker

Containerización de apps para que el entorno sea idéntico en dev y producción. Simplifica el deploy y elimina el clásico "en mi máquina funciona".

⚙️
GitHub Actions

CI/CD automatizado con caché de dependencias. Cada push dispara build, tests y deploy sin intervención manual.

⚡
Redis

Almacenamiento en memoria con TTL nativo para la expiración de notas en Snaps. Alta velocidad y cero overhead para datos temporales.
